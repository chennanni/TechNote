# Editor - IntelliJ

## Shortcut

~~~
Show Error message: Ctrl + F1
Auto Complete: Ctrl + Shift + Enter

Search for files: Shift + Shift + Shift
Search for word: Ctrl + Shift + R
Find usages of: Alt+F7
Show class hierarchy: Ctrl + H

Navigate to the implementation: Ctrl+Alt+B
Navigate to the super method: Ctrl+U

Go to definition: F4
Show method definition snapshot: Ctrl + Shift + I

Insert getter and setter: Alt + Insert

Switch files: Alt+Enter
~~~
